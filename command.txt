    npm run build   - запуск обычной сборки проекта 
    npm run build_avto  - запуск сборки с перезапуском в случае изменения в файлах

    npm run dev  - запуск webpack-dev-server в режиме - development
    npm run prod  - запуск webpack-dev-server в режиме - production
